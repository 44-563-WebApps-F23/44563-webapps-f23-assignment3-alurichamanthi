For a different code snippet, I found a relevant question on Stack Overflow regarding HTML and embedded video. Here's the question along with a quick-link to the source:

Question: How can I embed HTML5 video with a Flash fallback for cross-browser compatibility?

Source on Stack Overflow

Below is the code snippet for embedding HTML5 video with a Flash fallback:

        
            <!-- Video for Everybody - HTML5 Video with Flash fallback -->
            <video controls="controls" poster="path/to/poster.jpg" width="640" height="360">
                <source src="path/to/video.mp4" type="video/mp4" />
                <!-- Flash fallback for non-HTML5 browsers without JavaScript -->
                <object type="application/x-shockwave-flash" data="flashmediaelement.swf" width="640" height="360">
                    <param name="movie" value="flashmediaelement.swf" />
                    <param name="flashvars" value="controls=true&file=path/to/video.mp4" />
                    <!-- Image as a last resort -->
                    <img src="path/to/poster.jpg" width="640" height="360" alt="Video" title="No video playback capabilities" />
                </object>
            </video>
